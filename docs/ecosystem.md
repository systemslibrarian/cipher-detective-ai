# Ecosystem

Cipher Detective AI is part of a broader cryptography-education path:

| Project           | Role                                                                 |
|-------------------|----------------------------------------------------------------------|
| **Cipher Museum**  | Curated history of classical and modern ciphers — the *what* and *when*. |
| **Crypto Compare** | Side-by-side comparison of algorithms, properties, and trade-offs.   |
| **Crypto Lab**     | Hands-on experimentation with cryptographic primitives.              |
| **Cipher Detective AI** | This project — *how* weak ciphers leak patterns and how cryptanalysis finds them. |
| **Meow Decoder**   | A friendlier, playful entry point to the same ideas.                 |

Together they aim to take a learner from "ciphers are mysterious" to
"cryptography is engineering with honest threat models."
